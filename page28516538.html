Извлекать данные из разных файлов, баз данных и API
Например, список из поисковых запросов, по которым пользователи попадали на сайт. В этих запросах есть знаки препинания, лишние пробелы или же слова написаны
в разном регистре. При помощи Power Query можно настроить обработку данных так, чтобы получился однородный список для последующей обработки и классификации.
К примеру, файлы csc, xls, xml, json, базы данных SQL, MySQL, Postgress, BigQuery, api - Google Analytics, Яндекс.Метрика, Comagic

Например, вы сможете определить какие объявления вам необходимо переписать, какие ключевые слова остановить или уточнить, какие ставки нужно повысить или понизить.
Для прохождения курса вам понадобится компьютер
с Windows с оперативной памятью не менее 4GB, т.к.
Microsoft Power BI (пока) нет на другие ОС.
Если вы пользуетесь Mac, то вам потребуется Parallels.
По ссылке ниже есть шпаргалка для пользователей Mac.

Вывод справки по функциям Power Query

Matt Masson (Мэт Мэссон) и Theresa Palmer (Тереза Палмер) в своей недавней презентации Power Query and M for the PASS DW/BI Virtual Chapter показали одну интересную вещь – вывод справки по функциям Power Query с использованием строки формул.

Возьмём, например, функцию Text.Replace(). Создадим пустой запрос и введем в строку формул следующее:
= Text.Replace

Получим такой результат:
Сделав это, мы создаём шаг, который функция возвращает сама по себе. Обратите внимание, что это не то же самое, что вызов функции. Хотя её можно вызвать на следующем шаге. Если вам понадобится использовать функцию, кликните Invoke и введите данные для передачи в неё:
Нажимаем OK и получаем результат:
Вот весь код для тех, кто интересуется:
let
        Source = Text.Replace,
        InvokedSource = Source("the cat sat on the mat", "cat", "dog")
in
        InvokedSource
Очень полезный совет! К сожалению, нельзя задать справку для собственных функций, хотя Мэт говорил, что хотел бы иметь такую возможность.
